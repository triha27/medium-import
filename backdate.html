Setup A Collaborative Development Environment with Unity Cloud and Version Control

Goal

How to Set up your Plastic SCM and cloud build for your Unity project.

Audience

This is a guide for Unity developers of all levels who want to collaborate with a team to build a project together.

Result

Have your own build server running powered by Unity.

Assumption

At least a beginner-level familiarity with setting up projects in Unity via the Unity hub. You already have updated the Unity hub and downloaded the Unity editor (the editor is downloaded from the Unity hub).

Download Unity Hub: https://unity.com/download

Steps

Part A: Make a Unity ID

(Skip this part if you already have a Unity account)

  1. Go to the Unity website: https://unity.com/
  2. Click the profile icon and select "Create Unity ID."
  3. Follow the instructions on the page to create your ID and confirm your email.
  4. Sign in again to make sure the ID works (you don't need Unity advanced for cloud control).

Part B: Sign up for Unity DevOps/Create your Organization Server

  1. Go to https://dashboard.unity3d.com/ and sign in with your Unity account. This will open your Unity gaming dashboard.
  2. Find the section for creating a project, which should say, “Create a project to get started with gaming services.” Click on "Create Project" and enter the project name in the pop-up. (Skip this step if you already have projects)
  3. After you have a project, click on "DevOps" in the left navigation.
  4. You should see a page that says, “No Unity version control organizations yet.” Click on "Sign up now."
  5. This should open a new window for Unity DevOps. Complete the steps on top (Order > Your Info > Payment > Confirmation) to set up your Unity organization.
  6. Note: On the "Your Info" page, you will have an option to name your organization. By default, it's your email, but this can be changed.

Part C: Set up Plastic SCM on Your Unity Project

  1. Open Unity Hub and go to Settings > Preferences > Licenses, and make sure you have your respective license (Unity Free, Unity Pro, etc).
  2. Click on "Installs" and install the editor version of your choice.
  3. Now go to "Projects," click "New Project," and select a template of your choice.
  4. In the Unity editor, go to Window > Plastic SCM to open the Plastic SCM window.
  5. If you are not already signed in with your Unity ID, sign in and complete the first step in the SCM window.
  6. Next, you will be asked to select the organization and server. By default, it should select the organization you created with DevOps. (If you have multiple organizations, you can choose them by clicking the three dots beside the server.)
  7. Click "Create" to confirm, and the repository should be created and updated on the Unity Gaming Dashboard's DevOps section.
